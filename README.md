# Restapps Typescript

Restapps Typescript with Typeorm

## Feature

- Typescript
- ExpressJs
- Cloud Firestore

## Start Apps / Project

```
npm install

edit config .env

npm run start or nodemon
```

## Build Apps / Project

```
npm run build

```

## Guideline-API

```
1. sort => sort=ASC (sample) (ASC or DESC)
2. limit => default 10 => limit=10 (sample)
3. page => default 1 => page=1 (sample)


```
